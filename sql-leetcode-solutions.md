#### **1757 - Recyclable and Low Fat Products**
```markdown
SELECT product_id  
FROM Products  
WHERE low_fats = 'Y'  
AND recyclable = 'Y'  
````

#### **584. Find Customer Referee**

Select name from Customer  where referee_id != 2 OR referee_id is null
